DDR Controls:

Left (Green): Turns it left normally
Right (Green): Turns it right normally
Up (Blue): Drives Forwards
Left (Red): Swevre drives left
Right (Red): Swevre drives right
Down (Blue): Drives backwards

XBOX Controls:

Hold right trigger while using Xbox controls!
Standard arcade drive (Left Y [Forwards/Backwards] Right X [Left/Right])
Using an XBOX controller with the DDR bad will result in the code ignoring the DDR inputs and only listening to the Xbox Controller.

This is a safety feature

Hold left trigger to control the arm

Left Y: Raises and lowers the arm. Hold this down to initally lift the arm from rest position.

Right Y: Raises and lowers the forearm.

Left Toggle Button: Closes the hand

Right Toggle Button: Opens the hand

XBOX DDR Controls:

Since the DDR pad is recognized as a XBOX Controller, here are the coresponding controls!

These will be override with the arcade mode toggle!

These do NOT override DDR pad.

X Button: Forwards
B Button: Backwards
Y Button: Swerve Left
A Button: Swerve Right
Back: Turns it left normally
Start: Turns it right normally

Smart Dashboard:

This robot utilizes the Smart Dashboard. To access smart dashboard go to the settings icon in the FRC Driver station, and change Dashboard Type to SmartDashboard

Speed: Set's the speed of the robot when using the DDR pad. This value is out of 1, so 50% is 0.50

- For safety reasons, I reccomend keeping this value at or below 40-50% when using at outreach.

The other 2 values are data from the motor encoders, they're uneditable and can usually be disregarded.

Troubleshooting & Tips:

It is highly reccomended that you don't run the motor against a solid object, if the arm gets stick slowly manually raise it back to it's resting position.

If a motor stops moving, check the cables attached to it and the motor controller, if any of the red, white, or black cables are disconnected, turn off the robot and reattach them.

If a motor controller of a non working motor is blinking a strange color, use the rev hardware client to diagnose it.

To do that, plug the usb-c cable into a motor controller and use the icon next to the motor controllers to identify the motor controller, and look under sticky faults for the issue.
